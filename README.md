# 🗳️ AhilyaVote

> 🎓 Built for learning purposes only.

A full-stack voting & polling app where users can create polls and vote in real time.

---

## 🛠️ Tech Stack
| Layer | Technology |
|:------|:-----------|
| **Backend** | Java + Spring Boot |
| **Frontend** | Angular (TypeScript) |
| **Database** | MySQL + Spring Data JPA |
| **Build Tool** | Maven |

---

## ✨ Features
- **Create** polls with multiple options
- **Vote** on any active poll
- **View** poll results instantly
- **REST API** with Spring Boot
- **Angular** frontend

---

## 🚀 How to Run

**▶ Backend**
```bash
cd AhilyaVote
./mvnw spring-boot:run
```
**Runs on →** `http://localhost:8080`

**▶ Frontend**
```bash
cd VoteApp
npm install
ng serve
```
**Runs on →** `http://localhost:4200`
